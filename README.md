# OneTime Website And Privacy Policy

Standalone GitHub Pages site for the OneTime Chrome extension.

## Published URL

Use GitHub Pages from the `main` branch root. If the repository is named `onetime-privacy` under the
`Cashew-Milk` account, the published URLs are:

- Website: `https://cashew-milk.github.io/onetime-privacy/`
- Privacy policy: `https://cashew-milk.github.io/onetime-privacy/privacy/`

## Files

- `index.html`: public website page for the Chrome Web Store listing
- `privacy/index.html`: public privacy policy page for Chrome Web Store submission
