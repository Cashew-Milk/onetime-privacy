# OneTime Privacy Policy

Standalone privacy policy site for the OneTime Chrome extension.

## Published URL

Use GitHub Pages from the `main` branch root. If the repository is named `onetime-privacy` under the
`Cashew-Milk` account, the final policy URL is:

`https://cashew-milk.github.io/onetime-privacy/`

## Files

- `index.html`: public privacy policy page for Chrome Web Store submission
