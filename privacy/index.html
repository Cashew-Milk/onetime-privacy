OneTime

Privacy Policy

OneTime is a Chrome extension that helps users import hours from an Excel workbook and read, preview, fill, and delete Certinia PSA timecards. It handles only the information needed to provide those features.

Effective date: April 15, 2026

Information OneTime Handles

OneTime may handle the following information:

  • Spreadsheet data selected by the user, including project codes, ticket IDs, step or task names, comments, dates, and hour values.
  • Certinia page data needed to read, preview, fill, or delete timecards, including project names, task or milestone information, approval status, existing timecard values, and related page context.
  • Limited browsing context needed to confirm that the active tab is a supported Certinia page, such as the current tab's URL or domain.
  • Extension preferences and local state, such as theme, layout, onboarding state, imported session state, selected items, and local performance data.

How OneTime Uses Information

OneTime uses this information only to:

  • Parse the workbook selected by the user.
  • Show previews, validation messages, and local debug information.
  • Match workbook entries to Certinia projects and tasks.
  • Read, create, update, and delete timecard entries in the user's Certinia environment at the user's direction.
  • Store preferences and temporary session state locally so the extension continues working between popup openings.

How Information Is Stored and Shared

  • OneTime does not send user data to the developer.
  • OneTime does not sell, rent, or transfer user data to advertisers, data brokers, analytics providers, or other third parties.
  • OneTime does not use remote servers operated by the developer.
  • Data handled by OneTime remains local to the user's browser, except for communications with the user's own Certinia or Salesforce environment that are necessary to complete the actions requested by the user.

Data Retention

  • Preference data and certain local extension data may remain in Chrome extension storage until the user clears extension data or uninstalls the extension.
  • Temporary session data may be retained in Chrome session storage during the active browser session to preserve workflow state.

User Choices

  • Users choose whether to load a workbook into OneTime.
  • Users choose whether to run OneTime on a supported Certinia page.
  • Users can remove the extension or clear its stored data through Chrome's extension management tools.

Security

OneTime is designed to operate only on supported Certinia or Salesforce pages and to use Chrome extension storage for local state. The extension relies on the browser's and the user's Certinia environment's security controls for local processing and secure page communication.

Changes to This Policy

This privacy policy may be updated from time to time to reflect product changes. The current version will be posted at this URL.
